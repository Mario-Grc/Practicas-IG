# Prácticas IG
Este repositorio se ha creado para guardar las prácticas realizadas para la asignatura Informática Gráfica (IG) del Grado en Ingeniería Informática de la Universidad de Las Palmas de Gran Canaria (ULPGC), curso 2025-2026.

Estas prácticas están implementadas principalmente en un entorno web mediante **JavaScript**, **HTML5** y **WebGL**, haciendo uso de la biblioteca **Three.js** para facilitar la creación y manipulación de escenas 3D, cámaras, luces, materiales y animaciones. Además, se ha empleado **GLSL** para el desarrollo de shaders personalizados, principalmente en la práctica dedicada a shaders (Semana 9).
